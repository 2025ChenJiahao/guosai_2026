### 4.2 physics.py — 物理模型

核心物理计算模块，实现所有运动方程和遮蔽条件判断。

**函数列表：**

| 函数 | 输入 | 输出 | 说明 |
|------|------|------|------|
| `missile_pos(P_m0, t)` | 导弹初始位置, 时刻t | 导弹位置(3,) | 计算导弹t时刻位置 |
| `uav_pos(P_u0, v_u, theta, t)` | 无人机初始位置, 速度, 方向角, 时刻t | 无人机位置(3,) | 计算无人机t时刻位置 |
| `bomb_drop_point(P_u0, v_u, theta, t_d)` | 无人机参数, 投放时刻 | 投放点(3,) | 计算投放点坐标 |
| `bomb_detonate_point(P_u0, v_u, theta, t_d, delta_t)` | 无人机参数, 投放时刻, 起爆延迟 | 起爆点(3,) | 计算起爆点坐标 |
| `bomb_pos(P_u0, v_u, theta, t_d, t)` | 无人机参数, 投放时刻, 当前时刻 | 弹位置(3,) | 抛体运动阶段位置 |
| `cloud_center(P_e, t, t_e)` | 起爆点, 当前时刻, 起爆时刻 | 云团中心(3,) | 计算云团下沉位置 |
| `shielding_distance(P_m, P_c, T_target)` | 导弹位置, 云团中心, 目标位置 | 距离(m) | 云团到视线段距离 |
| `is_shielded(P_m, P_c, ...)` | 导弹位置, 云团中心 | bool | 是否有效遮蔽 |
| `sample_cylinder_surface(C, r, h, ...)` | 圆柱参数 | 采样点(N,3) | 圆柱体表面采样 |

**关键算法 — 遮蔽距离计算：**

```python
def shielding_distance(P_m, P_c, T_target):
    # P_m: 导弹位置, P_c: 云团中心, T_target: 目标位置
    v = T_target - P_m          # 视线方向向量
    w = P_c - P_m               # 云团到导弹向量
    s = dot(w,v) / dot(v,v)     # 投影参数
    s = clamp(s, 0, 1)          # 约束到[0,1]
    Q = P_m + s * v             # 视线上最近点
    return norm(P_c - Q)        # 云团到最近点距离
```

有效遮蔽条件：`shielding_distance <= R (10m)`

### 4.3 pso.py — 粒子群优化算法

实现标准PSO算法，支持引导初始化。

**类：`PSO`**

```python
PSO(n_particles, n_dims, lb, ub, fitness_func, seed_points=None)
```

- `n_particles`: 粒子数量
- `n_dims`: 决策变量维度
- `lb`, `ub`: 各维度上下界
- `fitness_func`: 适应度函数（越大越好）
- `seed_points`: 引导种子点列表，将部分粒子初始化在种子点附近

**核心参数：**
- 惯性权重 w: 0.9 → 0.4（线性递减）
- 学习因子 c1 = c2 = 2.0
- 速度限制: 各维度范围的20%

**引导初始化策略：**
将前 n_particles//3 个粒子初始化在种子点附近（5%范围随机扰动），确保PSO从有希望的区域开始搜索。
