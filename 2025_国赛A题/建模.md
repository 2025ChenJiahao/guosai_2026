# 2025年全国大学生数学建模竞赛A题：烟幕干扰弹投放策略

## 一、问题重述

### 1.1 问题背景

烟幕干扰弹通过化学燃烧或爆炸分散形成烟幕或气溶胶云团，在来袭武器和保护目标之间形成遮蔽，干扰敌方导弹的探测系统。本问题研究如何利用5架无人机（FY1-FY5）投放烟幕干扰弹，对3枚来袭导弹（M1、M2、M3）实施有效遮蔽，最大化对真目标的保护时间。

### 1.2 已知条件

**导弹参数：**
- 飞行速度：300 m/s
- 飞行方向：直指假目标（原点O）
- 初始位置：M1(20000, 0, 2000), M2(19000, 600, 2100), M3(18000, -600, 1900)

**无人机参数：**
- 速度范围：70~140 m/s
- 飞行方式：等高度匀速直线飞行
- 受领任务后可瞬时调整飞行方向
- 每架无人机投放两枚弹至少间隔1s
- 初始位置：FY1(17800, 0, 1800), FY2(12000, 1400, 1400), FY3(6000, -3000, 700), FY4(11000, 2000, 1800), FY5(13000, -2000, 1300)

**烟幕干扰弹参数：**
- 脱离无人机后在重力作用下运动（忽略空气阻力）
- 起爆后瞬时形成球状烟幕云团
- 云团以3 m/s匀速下沉
- 云团中心10m范围内烟幕浓度在起爆20s内可提供有效遮蔽

**目标参数：**
- 假目标：原点O(0, 0, 0)
- 真目标：圆柱形，半径7m，高10m，下底面圆心C(0, 200, 0)

### 1.3 五个子问题

1. **问题1**：FY1以120m/s朝向假目标飞行，1.5s后投放1枚弹，3.6s后起爆，求有效遮蔽时长
2. **问题2**：FY1投放1枚弹，优化飞行方向、速度、投放点、起爆点，最大化遮蔽时间
3. **问题3**：FY1投放3枚弹，设计投放策略
4. **问题4**：FY1、FY2、FY3各投放1枚弹，设计投放策略
5. **问题5**：5架无人机各至多投放3枚弹，干扰3枚导弹

## 二、问题分析

### 2.1 核心物理过程

烟幕干扰弹的完整运动过程分为三个阶段：

1. **随无人机飞行阶段**（t=0 ~ t_d）：弹随无人机以相同速度飞行
2. **抛体运动阶段**（t_d ~ t_e）：弹脱离无人机后在重力作用下做抛体运动
3. **云团下沉阶段**（t_e ~ t_e+20）：起爆后云团以3m/s匀速下沉

### 2.2 遮蔽条件的几何本质

**关键理解**：有效遮蔽不是指导弹距离云团近，而是**烟幕云团遮挡了导弹对真目标的视线**。

导弹飞行方向指向假目标（原点），但真目标在(0, 200, 0)。烟幕干扰弹的目的是在导弹和真目标之间形成遮蔽屏障，使导弹无法发现真目标。

因此，有效遮蔽的几何条件是：
- 云团中心到「导弹→真目标」视线段的距离 ≤ 10m（云团有效遮蔽半径）
- 且在起爆后20s时间窗口内

### 2.3 真目标的几何处理

真目标是圆柱体（半径7m，高10m），精确处理需要：
- 对圆柱体表面采样多个点
- 检查云团是否遮挡了导弹到所有这些点的视线

简化处理：用圆柱体中心点(0, 200, 5)近似代表真目标。对于本题的精度要求，这种简化是合理的。

## 三、模型假设

1. 忽略空气阻力对烟幕干扰弹运动的影响
2. 重力加速度取g = 9.8 m/s²，方向垂直向下
3. 烟幕干扰弹脱离无人机后立即进入抛体运动，无过渡阶段
4. 烟幕云团起爆后瞬时形成球状，且在整个有效期内保持球形
5. 烟幕云团的有效遮蔽范围为其中心周围10m的球形区域
6. 有效遮蔽时间为起爆后20s内

9. 不考虑地球曲率和大气折射的影响
10. 烟幕浓度在有效遮蔽范围内均匀分布

## 五、数学模型建立

### 5.1 坐标系

以假目标为原点，水平面为xy平面，z轴垂直向上。

### 5.2 导弹运动模型

导弹i的初始位置 $P_{m0}^i$，飞行方向指向原点O：

$$\vec{d}_m^i = \frac{O - P_{m0}^i}{|O - P_{m0}^i|} = \frac{(-x_{m0}^i, -y_{m0}^i, -z_{m0}^i)}{\sqrt{(x_{m0}^i)^2 + (y_{m0}^i)^2 + (z_{m0}^i)^2}}$$

导弹i在时刻t的位置：
$$P_m^i(t) = P_{m0}^i + v_m \cdot t \cdot \vec{d}_m^i$$

**各导弹方向向量计算：**

- M1: $\vec{d}_m^1 = \frac{(-20000, 0, -2000)}{\sqrt{20000^2+2000^2}} = \frac{(-20000, 0, -2000)}{20099.75} \approx (-0.9950, 0, -0.0995)$
- M2: $\vec{d}_m^2 = \frac{(-19000, -600, -2100)}{|(-19000, -600, -2100)|}$
- M3: $\vec{d}_m^3 = \frac{(-18000, 600, -1900)}{|(-18000, 600, -1900)|}$

### 5.3 无人机运动模型

无人机j的初始位置 $P_{u0}^j$，飞行方向 $\vec{d}_u^j = (\cos\theta_j, \sin\theta_j, 0)$（水平飞行），速度 $v_u^j \in [70, 140]$。

无人机j在时刻t的位置（t≥0，以受领任务为t=0时刻）：
$$P_u^j(t) = P_{u0}^j + v_u^j \cdot t \cdot \vec{d}_u^j$$

### 5.4 烟幕干扰弹运动模型

#### 5.4.1 投放阶段（随无人机飞行）

弹k由无人机j在时刻 $t_d^k$ 投放，投放点：
$$P_d^k = P_u^j(t_d^k) = P_{u0}^j + v_u^j \cdot t_d^k \cdot \vec{d}_u^j$$

#### 5.4.2 抛体运动阶段（投放→起爆）

弹k在时刻t（$t_d^k \leq t \leq t_e^k$）的位置：
$$P_b^k(t) = P_d^k + \vec{v}_0^k \cdot (t - t_d^k) + \frac{1}{2} \vec{g} (t - t_d^k)^2$$

其中：
- 初速度 $\vec{v}_0^k = v_u^j \cdot \vec{d}_u^j$（等于无人机速度）
- $\vec{g} = (0, 0, -g) = (0, 0, -9.8)$

起爆点：
$$P_e^k = P_b^k(t_e^k) = P_d^k + \vec{v}_0^k \cdot (t_e^k - t_d^k) + \frac{1}{2} \vec{g} (t_e^k - t_d^k)^2$$

展开：
$$P_e^k = P_{u0}^j + v_u^j \cdot t_d^k \cdot \vec{d}_u^j + v_u^j \cdot \vec{d}_u^j \cdot (t_e^k - t_d^k) + \frac{1}{2}(0,0,-g)(t_e^k - t_d^k)^2$$
$$= P_{u0}^j + v_u^j \cdot t_e^k \cdot \vec{d}_u^j + (0, 0, -\frac{1}{2}g(t_e^k - t_d^k)^2)$$

#### 5.4.3 云团下沉阶段（起爆后20s内）

云团k在时刻t（$t_e^k \leq t \leq t_e^k + 20$）的中心位置：
$$P_c^k(t) = P_e^k + (0, 0, -v_c) \cdot (t - t_e^k)$$
$$= (x_e^k, y_e^k, z_e^k - v_c(t - t_e^k))$$

### 5.5 有效遮蔽条件（核心，需修正）

#### 5.5.1 几何模型

**错误理解**：$|P_m(t) - P_c(t)| \leq 10$（导弹到云团距离≤10m）

**正确理解**：烟幕云团遮挡了导弹对真目标的视线。有效遮蔽条件为：

云团中心 $P_c(t)$ 到导弹 $P_m(t)$ 与真目标中心 $T = (0, 200, 5)$ 连线段的距离 $\leq R = 10$m。

#### 5.5.2 点到线段距离的计算

设导弹位置为M，真目标中心为T，云团中心为C。

视线段的参数方程：$\vec{L}(s) = M + s(T - M)$，$s \in [0, 1]$

令 $\vec{v} = T - M$，$\vec{w} = C - M$

参数：
$$s^* = \frac{\vec{w} \cdot \vec{v}}{\vec{v} \cdot \vec{v}}$$

约束到[0, 1]：
$$s^* = \max(0, \min(1, s^*))$$

最近点：$Q = M + s^* \cdot \vec{v}$

距离：
$$d(t) = |C - Q| = |P_c(t) - (P_m(t) + s^* \cdot (T - P_m(t)))|$$

#### 5.5.3 有效遮蔽判据

$$\text{Shields}(t) = \begin{cases} 1, & \text{if } d(t) \leq R \text{ and } t_e \leq t \leq t_e + T_{eff} \\ 0, & \text{otherwise} \end{cases}$$

#### 5.5.4 考虑真目标为圆柱体的精确模型

真目标是圆柱体，精确遮蔽需要检查导弹到圆柱体表面上任意点的视线是否被遮挡。简化方案：

**方案A（点目标近似）**：用圆柱体中心点 $T = (0, 200, 5)$ 代表真目标。

**方案B（多点采样）**：在圆柱体表面均匀采样N个点 $\{T_1, T_2, ..., T_N\}$，对每个点计算遮蔽条件。有效遮蔽要求对所有采样点都被遮挡：

$$\text{Shields}(t) = \begin{cases} 1, & \text{if } \max_{k=1..N} d_k(t) \leq R \\ 0, & \text{otherwise} \end{cases}$$

**推荐**：采用方案B，在圆柱体侧面和上下底面各取若干点（共约20-50个点），既保证精度又不过于复杂。

### 5.6 优化目标

**单弹遮蔽时长**：
$$T_{shield}^k = \int_{t_e^k}^{t_e^k + T_{eff}} \mathbb{1}[d^k(t) \leq R] \, dt$$

**总遮蔽时长**（多弹联合，可不连续）：
$$T_{total} = \text{measure}\left(\bigcup_{k=1}^{N} \{t : d^k(t) \leq R, \, t_e^k \leq t \leq t_e^k + T_{eff}\}\right)$$

其中measure表示集合的Lebesgue测度（即并集的总长度）。

**对于问题5（多导弹）**：
$$T_{total} = \sum_{i=1}^{3} T_{total}^i$$

或取最小值（木桶效应）：
$$T_{total} = \min_{i=1,2,3} T_{total}^i$$

**推荐**：采用求和形式，因为题目要求"对M1、M2、M3等3枚来袭导弹的干扰"，应最大化对所有导弹的总遮蔽时间。

## 六、各问题详细求解

### 6.1 问题1：FY1投放1枚弹干扰M1

**已知条件：**
- FY1初始位置：$P_{u0} = (17800, 0, 1800)$
- 飞行速度：$v_u = 120$ m/s
- 飞行方向：朝向假目标（原点）
- 投放时刻：$t_d = 1.5$ s
- 起爆延迟（投放后）：3.6 s，即 $t_e = 1.5 + 3.6 = 5.1$ s

**Step 1：计算FY1飞行方向**

FY1朝向假目标飞行，但需保持高度不变。方向向量在水平面上的投影：
$$\vec{d}_u = \frac{(0-17800, 0-0, 0)}{|(0-17800, 0-0, 0)|} = (-1, 0, 0)$$

**Step 2：计算投放点**
$$P_d = (17800, 0, 1800) + 120 \times 1.5 \times (-1, 0, 0) = (17620, 0, 1800)$$

**Step 3：计算起爆点**

弹的初速度 = 无人机速度 = $(-120, 0, 0)$ m/s

从投放到起爆的时间间隔：$\Delta t = t_e - t_d = 3.6$ s

$$P_e = (17620, 0, 1800) + (-120, 0, 0) \times 3.6 + \frac{1}{2}(0, 0, -9.8) \times 3.6^2$$
$$= (17620 - 432, 0, 1800 - 63.504) = (17188, 0, 1736.496)$$

**Step 4：计算导弹M1轨迹**

M1初始位置：$(20000, 0, 2000)$

方向向量：$\vec{d}_m = \frac{(-20000, 0, -2000)}{20099.75} \approx (-0.9950, 0, -0.0995)$

$$P_m(t) = (20000 - 298.5t, \, 0, \, 2000 - 29.85t)$$

其中 $v_m \cdot \vec{d}_m = 300 \times (-0.9950, 0, -0.0995) = (-298.5, 0, -29.85)$

**Step 5：计算云团轨迹**

$$P_c(t) = (17188, 0, 1736.496) + (0, 0, -3)(t - 5.1)$$
$$= (17188, 0, 1736.496 - 3t + 15.3) = (17188, 0, 1751.796 - 3t)$$

**Step 6：计算遮蔽距离**

真目标中心：$T = (0, 200, 5)$

导弹位置：$M(t) = (20000 - 298.5t, 0, 2000 - 29.85t)$

云团中心：$C(t) = (17188, 0, 1751.796 - 3t)$

视线方向：$\vec{v}(t) = T - M(t) = (-20000 + 298.5t, 200, -1995 + 29.85t)$

$\vec{w}(t) = C(t) - M(t) = (17188 - 20000 + 298.5t, 0, 1751.796 - 3t - 2000 + 29.85t)$
$= (-2812 + 298.5t, 0, -248.204 + 26.85t)$

$$s^*(t) = \frac{\vec{w}(t) \cdot \vec{v}(t)}{\vec{v}(t) \cdot \vec{v}(t)}$$

最近点距离：
$$d(t) = |\vec{w}(t) - s^*(t) \cdot \vec{v}(t)|$$

**Step 7：数值求解有效遮蔽时长**

在 $t \in [5.1, 25.1]$ 范围内，以 $\Delta t = 0.01$ s 步长遍历，统计 $d(t) \leq 10$ 的时间。

**Python实现要点：**
```python
import numpy as np

# 定义各轨迹函数
def missile_pos(t):
    return np.array([20000 - 298.5*t, 0, 2000 - 29.85*t])

def cloud_pos(t):
    return np.array([17188, 0, 1751.796 - 3*t])

def shield_distance(t):
    M = missile_pos(t)
    C = cloud_pos(t)
    T = np.array([0, 200, 5])
    v = T - M
    w = C - M
    s = np.dot(w, v) / np.dot(v, v)
    s = max(0, min(1, s))
    Q = M + s * v
    return np.linalg.norm(C - Q)

# 遍历计算
dt = 0.01
times = np.arange(5.1, 25.1, dt)
distances = [shield_distance(t) for t in times]
shield_time = sum(dt for d in distances if d <= 10)
print(f"有效遮蔽时长: {shield_time:.2f} s")
```

### 6.3 问题3：FY1投放3枚弹

**决策变量（共3组+2个共享变量）：**
- 共享：飞行方向 $\theta$，飞行速度 $v_u$
- 第i枚弹：投放时刻 $t_{d,i}$，起爆延迟 $\Delta t_i$（i=1,2,3）

**约束条件：**
1. $70 \leq v_u \leq 140$
2. $t_{d,i+1} \geq t_{d,i} + 1$（投放间隔≥1s）
3. $\Delta t_i \geq 0$
4. 总决策变量数：$2 + 3 \times 2 = 8$ 个

**优化目标：**
$$\max T_{total} = \text{measure}\left(\bigcup_{i=1}^{3} [t_{e,i}, t_{e,i}+20] \cap \{t: d_i(t) \leq R\}\right)$$

**求解策略：**

由于3枚弹的遮蔽时间窗口可能重叠，需要计算并集。采用以下策略：

1. **PSO搜索全局最优**：8维搜索空间
2. **适应度计算**：对每枚弹分别计算遮蔽时间区间，再求并集长度
3. **策略建议**：
   - 第1枚弹：尽早投放，拦截导弹早期
   - 第2枚弹：在第1枚弹遮蔽结束前投放，实现无缝衔接
   - 第3枚弹：作为补充，填补空隙

**Python实现要点：**

```python
        intervals.append(get_shield_intervals(t_d, dt, theta, v_u))
    
    # 求并集长度
    return union_length(intervals)
```

### 6.5 问题5：5架无人机干扰3枚导弹

**问题规模：**
- 5架无人机，每架至多3枚弹 → 最多15枚弹
- 3枚导弹需同时干扰
- 决策变量：每架无人机的方向、速度 + 每枚弹的投放时刻和起爆延迟

**决策变量总数：**
- 5架无人机：$5 \times 2 = 10$ 个（方向、速度）
- 最多15枚弹：$15 \times 2 = 30$ 个（投放时刻、起爆延迟）
- 总计：最多40个决策变量

**约束条件：**
1. 同一无人机投放间隔≥1s
2. $70 \leq v_i \leq 140$
3. 每架无人机至多3枚弹

**优化目标：**
$$\max \sum_{j=1}^{3} T_{total}^j$$

其中 $T_{total}^j$ 是对导弹j的总遮蔽时长。

**求解策略：**

40维优化问题，PSO直接搜索效率低。建议：

1. **分层优化**：
   - 第一层：分配任务（哪架无人机干扰哪枚导弹）
   - 第二层：对每枚导弹，优化分配给它的无人机的投放策略

2. **任务分配策略**：
   - 根据无人机与导弹的几何关系，合理分配
   - FY1(17800,0,1800)离M1最近 → 干扰M1
   - FY4(11000,2000,1800)离M2较近 → 干扰M2
   - FY5(13000,-2000,1300)离M3较近 → 干扰M3
   - FY2、FY3作为补充

3. **遗传算法（GA）**：
   - 编码：(无人机编号, 干扰哪枚导弹, 飞行方向, 飞行速度, 投放时刻×3, 起爆延迟×3)
   - 适应度：对3枚导弹的总遮蔽时长
   - 交叉、变异操作

### 7.1 核心函数库

```python
import numpy as np
from scipy.optimize import minimize

import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D

# ============ 基础物理计算函数 ============

def missile_trajectory(P_m0, t):
    """计算导弹在时刻t的位置"""
    v_m = 300
    d_m = -P_m0 / np.linalg.norm(P_m0)  # 方向指向原点
    return P_m0 + v_m * t * d_m

def uav_trajectory(P_u0, v_u, theta, t):
    """计算无人机在时刻t的位置"""
    d_u = np.array([np.cos(theta), np.sin(theta), 0])
    return P_u0 + v_u * t * d_u

def bomb_drop_point(P_u0, v_u, theta, t_d):
    """计算投放点"""
    return uav_trajectory(P_u0, v_u, theta, t_d)

def bomb_detonate_point(P_u0, v_u, theta, t_d, delta_t):
    """计算起爆点"""
    P_d = bomb_drop_point(P_u0, v_u, theta, t_d)
    d_u = np.array([np.cos(theta), np.sin(theta), 0])
    v0 = v_u * d_u  # 初速度
    t_flight = delta_t  # 投放到起爆的时间
    g = np.array([0, 0, -9.8])
    return P_d + v0 * t_flight + 0.5 * g * t_flight**2

def cloud_center(P_e, t, t_e):
    """计算云团中心在时刻t的位置"""
    v_c = 3
    return P_e + np.array([0, 0, -v_c]) * (t - t_e)

def shielding_distance(P_m, P_c, T_target):
    """计算云团中心到导弹-目标视线段的距离"""
    v = T_target - P_m
    w = P_c - P_m
    vv = np.dot(v, v)
    if vv < 1e-10:
        return np.linalg.norm(P_c - P_m)
    s = np.dot(w, v) / vv
    s = max(0, min(1, s))
    Q = P_m + s * v
    return np.linalg.norm(P_c - Q)

def compute_shield_duration(P_u0, v_u, theta, t_d, delta_t, P_m0, T_target, 
                            R=10, T_eff=20, dt=0.01):
    """计算单枚弹的有效遮蔽时长"""
    t_e = t_d + delta_t
    P_e = bomb_detonate_point(P_u0, v_u, theta, t_d, delta_t)
    
    total = 0
    t = t_e
    while t <= t_e + T_eff:
        P_m = missile_trajectory(P_m0, t)
        P_c = cloud_center(P_e, t, t_e)
        d = shielding_distance(P_m, P_c, T_target)
        if d <= R:
            total += dt
        t += dt
    return total

# ============ 圆柱体目标处理 ============

def sample_cylinder_surface(C_center, r, h, n_side=16, n_top=8):
    """在圆柱体表面采样点"""
    points = []
    # 侧面
    for i in range(n_side):
        angle = 2 * np.pi * i / n_side
        for j in range(3):
            z = h * j / 2
            points.append([
                C_center[0] + r * np.cos(angle),
                C_center[1] + r * np.sin(angle),
                C_center[2] + z
            ])
    # 上底面
    for i in range(n_top):
        angle = 2 * np.pi * i / n_top
        for rho in [0, r/2, r]:
            points.append([
                C_center[0] + rho * np.cos(angle),
                C_center[1] + rho * np.sin(angle),
                C_center[2] + h
            ])
    # 下底面
    for i in range(n_top):
        angle = 2 * np.pi * i / n_top
        for rho in [0, r/2, r]:
            points.append([
                C_center[0] + rho * np.cos(angle),
                C_center[1] + rho * np.sin(angle),
                C_center[2]
            ])
    return np.array(points)

def compute_shield_duration_cylinder(P_u0, v_u, theta, t_d, delta_t, P_m0, 
                                      target_points, R=10, T_eff=20, dt=0.01):
    """考虑圆柱体目标的遮蔽时长计算"""
    t_e = t_d + delta_t
    P_e = bomb_detonate_point(P_u0, v_u, theta, t_d, delta_t)
    
    total = 0
    t = t_e
    while t <= t_e + T_eff:
        P_m = missile_trajectory(P_m0, t)
        P_c = cloud_center(P_e, t, t_e)
        
        # 检查是否对所有采样点都被遮挡
        all_shielded = True
        for T in target_points:
            d = shielding_distance(P_m, P_c, T)
            if d > R:
                all_shielded = False
                break
        
        if all_shielded:
            total += dt
        t += dt
    return total
```

### 7.2 PSO优化器

```python
class PSO:
    """粒子群优化器"""
    def __init__(self, n_particles, n_dims, lb, ub, fitness_func):
        self.n = n_particles
        self.d = n_dims
        self.lb = np.array(lb)
        self.ub = np.array(ub)
        self.fitness = fitness_func
        
        # 初始化粒子位置和速度
        self.positions = np.random.uniform(lb, ub, (n_particles, n_dims))
        self.velocities = np.zeros((n_particles, n_dims))
        
        # 个体最优和全局最优
        self.pbest_pos = self.positions.copy()
        self.pbest_val = np.full(n_particles, -np.inf)
        self.gbest_pos = None
        self.gbest_val = -np.inf
        
        # PSO参数
        self.w_max = 0.9
        self.w_min = 0.4
        self.c1 = 2.0
        self.c2 = 2.0
    
    def optimize(self, max_iter):
        convergence = []
        for iteration in range(max_iter):
            w = self.w_max - (self.w_max - self.w_min) * iteration / max_iter
            
            for i in range(self.n):
                # 计算适应度
                fit = self.fitness(self.positions[i])
                
                # 更新个体最优
                if fit > self.pbest_val[i]:
                    self.pbest_val[i] = fit
                    self.pbest_pos[i] = self.positions[i].copy()
                
                # 更新全局最优
                if fit > self.gbest_val:
                    self.gbest_val = fit
                    self.gbest_pos = self.positions[i].copy()
            
            convergence.append(self.gbest_val)
            
            # 更新速度和位置
            r1 = np.random.random((self.n, self.d))
            r2 = np.random.random((self.n, self.d))
            
            self.velocities = (w * self.velocities 
                             + self.c1 * r1 * (self.pbest_pos - self.positions)
                             + self.c2 * r2 * (self.gbest_pos - self.positions))
            
            self.positions += self.velocities
            
            # 边界约束
            self.positions = np.clip(self.positions, self.lb, self.ub)
            
            if iteration % 50 == 0:
                print(f"Iteration {iteration}: Best fitness = {self.gbest_val:.4f}")
        
        return self.gbest_pos, self.gbest_val, convergence
```

## 十、模型评价

### 10.1 模型优点

1. **物理建模准确**：完整考虑了弹的三阶段运动过程
2. **遮蔽条件合理**：基于视线遮挡的几何模型，而非简单的距离判据
3. **目标处理精细**：考虑了圆柱体目标的多点采样
4. **算法选择恰当**：PSO适合连续优化，GA适合离散组合优化
5. **可扩展性好**：模型框架可方便地扩展到更多无人机和导弹

### 10.2 模型缺点

1. **忽略了空气阻力**：实际弹道会有偏差
2. **未考虑风速影响**：烟幕云团可能被风吹散
3. **计算量较大**：圆柱体多点采样增加了计算复杂度
4. **PSO可能陷入局部最优**：需要多次运行或采用改进PSO

### 10.3 改进方向

1. 引入空气阻力修正抛体运动模型
2. 考虑风速对云团运动的影响
3. 采用自适应PSO或混合算法提高搜索效率
4. 增加蒙特卡洛模拟评估策略的鲁棒性
