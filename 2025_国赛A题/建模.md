# 2025年全国大学生数学建模竞赛A题：烟幕干扰弹投放策略

## 一、问题分析与理解

### 1.1 问题背景

烟幕干扰弹通过化学燃烧或爆炸分散形成烟幕或气溶胶云团，在目标前方特定空域形成遮蔽，干扰敌方导弹。本问题研究如何利用无人机投放烟幕干扰弹，最大程度地遮蔽来袭导弹对真目标的探测。

### 1.2 关键参数

| 参数 | 数值 |
|------|------|
| 导弹飞行速度 | 300 m/s |
| 无人机速度范围 | 70~140 m/s |
| 烟幕云团下沉速度 | 3 m/s |
| 有效遮蔽范围 | 云团中心10m内 |
| 有效遮蔽时间 | 起爆后20s内 |
| 两枚弹最小投放间隔 | 1 s |

### 1.3 目标与导弹位置

- **假目标**：原点 O(0, 0, 0)
- **真目标**：圆柱形，半径7m，高10m，下底面圆心 C(0, 200, 0)
- **导弹位置**：
  - M1(20000, 0, 2000)
  - M2(19000, 600, 2100)
  - M3(18000, -600, 1900)
- **无人机位置**：
  - FY1(17800, 0, 1800)
  - FY2(12000, 1400, 1400)
  - FY3(6000, -3000, 700)
  - FY4(11000, 2000, 1800)
  - FY5(13000, -2000, 1300)

## 二、数学模型建立

### 2.1 坐标系定义

- 原点：假目标位置
- x轴：水平面内
- y轴：水平面内，指向真目标方向
- z轴：垂直向上

### 2.2 导弹运动模型

导弹以速度 $v_m = 300$ m/s 直飞假目标。

导弹初始位置 $P_{m0} = (x_{m0}, y_{m0}, z_{m0})$，假目标位置 $O = (0, 0, 0)$。

导弹运动方向向量：
$$\vec{d}_m = \frac{O - P_{m0}}{|O - P_{m0}|} = \frac{(-x_{m0}, -y_{m0}, -z_{m0})}{\sqrt{x_{m0}^2 + y_{m0}^2 + z_{m0}^2}}$$

导弹在时刻 $t$ 的位置：
$$P_m(t) = P_{m0} + v_m \cdot t \cdot \vec{d}_m$$

### 2.3 无人机运动模型

无人机以等高度匀速直线飞行，速度范围 $v_u \in [70, 140]$ m/s。

无人机初始位置 $P_{u0} = (x_{u0}, y_{u0}, z_{u0})$，飞行方向单位向量 $\vec{d}_u = (\cos\alpha, \cos\beta, 0)$（保持高度不变）。

无人机在时刻 $t$ 的位置：
$$P_u(t) = P_{u0} + v_u \cdot t \cdot \vec{d}_u$$

约束条件：$z_{u0} + v_u \cdot t \cdot 0 = z_{u0}$（高度不变）

### 2.4 烟幕干扰弹运动模型

#### 2.4.1 投放阶段（抛体运动）

烟幕干扰弹在时刻 $t_d$ 从无人机位置 $P_d = P_u(t_d)$ 投放，初速度等于无人机速度 $\vec{v}_0 = v_u \cdot \vec{d}_u$。

在重力作用下（忽略空气阻力），烟幕弹在时刻 $t$（$t \geq t_d$）的位置：
$$P_b(t) = P_d + \vec{v}_0 \cdot (t - t_d) + \frac{1}{2} \vec{g} (t - t_d)^2$$

其中 $\vec{g} = (0, 0, -9.8)$ m/s²。

#### 2.4.2 起爆阶段

烟幕弹在时刻 $t_e$ 起爆，起爆点位置 $P_e = P_b(t_e)$。

#### 2.4.3 烟幕云团运动阶段

起爆后，烟幕云团以 $v_c = 3$ m/s 匀速下沉。

云团中心在时刻 $t$（$t \geq t_e$）的位置：
$$P_c(t) = P_e + (0, 0, -v_c) \cdot (t - t_e)$$

### 2.5 有效遮蔽条件

烟幕云团对导弹的有效遮蔽需同时满足：

1. **空间条件**：导弹轨迹与云团的距离 $d(t) \leq 10$ m
2. **时间条件**：$t_e \leq t \leq t_e + 20$ s

其中 $d(t)$ 为导弹在时刻 $t$ 的位置到云团中心的距离：
$$d(t) = |P_m(t) - P_c(t)|$$

### 2.6 优化目标

最大化有效遮蔽时长 $T$：
$$T = \int_{t_e}^{t_e+20} \mathbb{1}[d(t) \leq 10] \, dt$$

其中 $\mathbb{1}[\cdot]$ 为示性函数。

## 三、问题求解

### 3.1 问题1：FY1投放1枚弹干扰M1

**已知条件：**
- FY1初始位置：$P_{u0} = (17800, 0, 1800)$
- 飞行速度：$v_u = 120$ m/s
- 飞行方向：朝向假目标
- 投放延迟：$t_d = 1.5$ s
- 起爆延迟：$t_e = t_d + 3.6 = 5.1$ s

**求解步骤：**

1. **计算FY1飞行方向**：
   $$\vec{d}_u = \frac{(0, 0, 0) - (17800, 0, 1800)}{|(0, 0, 0) - (17800, 0, 1800)|} = \frac{(-17800, 0, -1800)}{\sqrt{17800^2 + 1800^2}}$$

   由于保持高度不变，忽略z分量：
   $$\vec{d}_u = (-1, 0, 0)$$

2. **计算投放点**：
   $$P_d = P_{u0} + v_u \cdot t_d \cdot \vec{d}_u = (17800, 0, 1800) + 120 \cdot 1.5 \cdot (-1, 0, 0) = (17620, 0, 1800)$$

3. **计算起爆点**：
   $$P_e = P_d + \vec{v}_0 \cdot (t_e - t_d) + \frac{1}{2} \vec{g} (t_e - t_d)^2$$
   $$= (17620, 0, 1800) + (-120, 0, 0) \cdot 3.6 + \frac{1}{2} (0, 0, -9.8) \cdot 3.6^2$$
   $$= (17620 - 432, 0, 1800 - 63.504) = (17188, 0, 1736.496)$$

4. **计算导弹轨迹**：
   M1初始位置：$P_{m0} = (20000, 0, 2000)$
   $$\vec{d}_m = \frac{(-20000, 0, -2000)}{\sqrt{20000^2 + 2000^2}} \approx (-0.9950, 0, -0.0995)$$

   导弹位置：
   $$P_m(t) = (20000 - 298.5t, 0, 2000 - 29.85t)$$

5. **计算云团轨迹**：
   $$P_c(t) = (17188 - 3(t - 5.1), 0, 1736.496 - 3(t - 5.1))$$
   $$= (17203.3 - 3t, 0, 1751.796 - 3t)$$

6. **计算有效遮蔽时长**：
   需要找到 $t \in [5.1, 25.1]$ 使得 $|P_m(t) - P_c(t)| \leq 10$

   距离函数：
   $$d(t) = \sqrt{(20000 - 298.5t - 17203.3 + 3t)^2 + (2000 - 29.85t - 1751.796 + 3t)^2}$$
   $$= \sqrt{(2796.7 - 295.5t)^2 + (248.204 - 26.85t)^2}$$

   求解 $d(t) \leq 10$，得到有效遮蔽时长。

### 3.2 问题2：优化FY1投放策略

**决策变量：**
- 飞行方向：$\vec{d}_u = (\cos\alpha, \cos\beta, 0)$
- 飞行速度：$v_u \in [70, 140]$ m/s
- 投放时刻：$t_d \geq 0$
- 起爆时刻：$t_e \geq t_d$

**优化模型：**
$$\max_{v_u, \alpha, t_d, t_e} T = \int_{t_e}^{t_e+20} \mathbb{1}[d(t) \leq 10] \, dt$$

**约束条件：**
1. $v_u \in [70, 140]$
2. $t_d \geq 0$
3. $t_e \geq t_d$
4. 无人机保持高度不变

**求解方法：**
采用数值优化方法（如粒子群算法或遗传算法）搜索最优参数组合。

## 四、算法设计

### 4.1 粒子群优化算法（PSO）

用于求解非线性优化问题。

**算法流程：**
1. 初始化粒子群（随机生成决策变量）
2. 评估适应度（有效遮蔽时长）
3. 更新个体最优和全局最优
4. 更新粒子速度和位置
5. 重复直到收敛

### 4.2 蒙特卡洛模拟

用于评估随机策略的有效遮蔽时长。

### 4.3 数值积分

用于计算有效遮蔽时长：
$$T \approx \sum_{k=0}^{N-1} \mathbb{1}[d(t_k) \leq 10] \cdot \Delta t$$

其中 $t_k = t_e + k \cdot \Delta t$，$\Delta t$ 为时间步长。
